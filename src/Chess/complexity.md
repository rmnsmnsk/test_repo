Оценка сложности:

1) Переборное решение
Всего мы можем в каждую строку поставить n чисел, таких строк n штук, Т.е всего n * n вариантов
Для каждого варианта вычисляется по две функции, каждая функция счиает сначала n, затем (n-1) .... 1
Таким образом по формуле арифм прогрессии всего вычисляется n * (n - 1)/2
т.к функции две, то получается n * (n - 1) ~ 0(n**2)
Таким обращом общая сложность - это 0(n ** n * n ** 2) = O(n ** (n + 2))

2) Рекурсивный способ
Неверные варианты отсекаются сразу, получается всего n!, при этом для каждой перестановки вычисляется ещё две функции, общей сложностью О(n**2)
Таким образом получается О(n! * n ** 2) - самый худший случай, если переставилось n элементов

3) Третье решение 

Аналогично рекурсии, сложность получается О(n! * n ** 2). Также достигаетс за счёт отсеивания неверных вариантов + каждый раз проверка в двух функциях общей сложнстью О(n ** 2)
